Create your own personal website.

Start with the basics..

  1. GitHub Account - Follow the link to setup a Github account:Github Signup Give yourself a nice username, which you will use in a later step.
  2. Create a new repository following the included link. New Repository Give the new repository the following name using your username in its place. username.github.io Ensure the repository is set to public and select "Create Repository".
  3. Navigate to the repository you just created. You can also navigate to its using the following address. https://github.com/username/username.github.io
  4. You will now need to install an html editor to create and edit the personal page. Download and install VS Code or a similar html editor application. Download VS Code

How to start the basics.

  1. Choose a location in which you will store the local files. Name the folder whatever you'd like.
  2. In VS Code or similar editor navigate to the folder you created and create a new file, html file. In VS Code you do this by selecting the new file icon, naming it like this "filename.html".
  3. Choose a profile picture you would like to display in the page, make sure that it is JPG format for this application. Paste a copy in the folder you just created. Name the file something simple to be referenced later.
  4. Open the file you just created and insert the following text, in the following format.
    5. <!DOCTYPE html>
    <html>
    <body>
    <h1>Heading Text</h1>
    <img src="./mypic.jpg"/>
    <p>Paragraph Text</p>
    </body>
    </html>
  5. Now that the document has its layout its time to add some of your own personal information. In the file you will want to replace "Heading Text" with a title for the portfolio page. Something like My Portfolio. Replace the "Paragraph Text" with some information about you. Use <b> within your text to create a line break.
  6. On the line <img src="./mypic.jpg" /> replace the "mypic" text with the name of your profile image. This will link its location and allow it to be displayed.
  7. After you are complete, your finished html file should look something like this.
    8. <!DOCTYPE html>
    <html>
    <body>
    <h1>Mei's Portfolio</h1>
    <img src="./meispic.jpg"/>
    <p>Hi my name is Mei.</p>
    </body>
    </html>
  8. Last thing to do is move the complete files to your github repository. Navigate the newly created repository. Select "upload an existing file", upload the html file and portfolio image you placed in your local file folder.

Enjoy your work.

  1. Now that everything is uploaded to the cloud you should be able to navigate to you new personal portfolio via a browser. Navigate to https://username/username.github.io
