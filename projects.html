Eyes Exercise

This is my Eyes Exercise. The two large eyes on the screen will follow your cursor as you move about. This was coded using html and javascript. Kinda creepy, kinda fun.

PacMen Exercise

This is my little PacMen Exercise. This is written using html and javascript. Loading the page you have the option to add PacMen, one by one. "Start Game" starts the movement of all the objects. Each PacMen moves in a direction and reverses direction upon collision with the page limits.

Real-Time Bus Tracker Exercise

This is my little Real-Time Bus Tracker Exercise. This project uses a google based API to create the map. This also uses real-time Mapbox data to track a moving bus through its normal route. Adds a pointer to the map and updates every 15sec to allow for constraints google/mapbox has on refresh rates.
